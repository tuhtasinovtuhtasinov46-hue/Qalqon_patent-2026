# Публикация на GitHub Pages

## Вариант через сайт GitHub

1. Откройте https://github.com/new
2. Название репозитория: `patent-mini-app`
3. Выберите `Public`.
4. Нажмите `Create repository`.
5. На странице репозитория нажмите `uploading an existing file`.
6. Загрузите все файлы из папки `github-pages-upload`.
7. Нажмите `Commit changes`.
8. Откройте `Settings` -> `Pages`.
9. В `Build and deployment` выберите:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
10. Сохраните. Через 1-3 минуты появится HTTPS-ссылка.

Ссылка будет примерно такой:

`https://ВАШ_ЛОГИН.github.io/patent-mini-app/`

Эту ссылку вставьте в BotFather как URL Mini App.
