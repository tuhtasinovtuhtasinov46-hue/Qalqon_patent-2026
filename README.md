# Telegram Mini App: Патент 2026

Готовый статический тренажер для подготовки к комплексному экзамену на патент.

## Что внутри

- режим пробного экзамена на 22 задания;
- тренировка по модулям: русский язык, история России, основы законодательства РФ;
- сохранение изученных вопросов, ошибок и лучшего результата в браузере;
- интерфейс адаптирован под Telegram Mini App и обычный мобильный браузер.

## Как открыть

Откройте `index.html` в браузере. Для публикации как Telegram Mini App разместите папку на HTTPS-хостинге и укажите URL в BotFather.

## Примечание

Вопросы являются авторским учебным набором по типовым темам экзамена. Это не официальный банк заданий.

## Использованные ориентиры

- rf-test.ru: комплексный экзамен включает русский язык, историю России и основы законодательства РФ.
- Регламент ТГУ им. Г. Р. Державина 2026: для разрешения на работу или патента указаны 22 задания, проходной ориентир 11 баллов всего, минимум 6 по русскому языку, 2 по истории и 3 по законодательству.
